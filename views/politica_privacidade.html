Política de Privacidade

Bem-vindo à nossa Política de Privacidade. Esta página informa sobre nossas políticas relativas à coleta, uso e divulgação de dados pessoais quando você utiliza nosso serviço e as escolhas que você tem associadas a esses dados.

1. Coleta e Uso de Informações

Coletamos vários tipos diferentes de informações para diversos fins, a fim de fornecer e melhorar nosso serviço para você.

Tipos de Dados Coletados

  • Dados Pessoais: Nome, endereço de e-mail, número de telefone, endereço, dados de uso e cookies.
  • Dados de Uso: Coletamos automaticamente informações sobre como o serviço é acessado e usado.
  • Cookies: Utilizamos cookies e tecnologias similares para rastrear atividades em nosso serviço.

2. Uso dos Dados

Utilizamos os dados coletados para diversas finalidades:

  • Fornecer e manter nosso serviço
  • Notificar sobre mudanças em nosso serviço
  • Permitir a participação em recursos interativos
  • Fornecer suporte ao cliente
  • Monitorar o uso do serviço
  • Detectar e prevenir problemas técnicos

3. Transferência de Dados

Suas informações, incluindo Dados Pessoais, podem ser transferidas para — e mantidas em — computadores localizados fora do seu estado, província, país ou outra jurisdição governamental onde as leis de proteção de dados podem ser diferentes daquelas da sua jurisdição.

4. Divulgação de Dados

Podemos divulgar seus Dados Pessoais nas seguintes situações:

  • A prestadores de serviços para monitorar e analisar o uso de nosso serviço
  • Para fins de aquisição de negócios, fusão ou venda de ativos
  • Para cumprir com obrigações legais ou proteger contra responsabilidade legal

5. Segurança dos Dados

A segurança dos seus dados é importante para nós, mas lembre-se que nenhum método de transmissão pela Internet ou método de armazenamento eletrônico é 100% seguro. Embora nos esforcemos para usar meios comercialmente aceitáveis para proteger seus Dados Pessoais, não podemos garantir sua segurança absoluta.

6. Links para Outros Sites

Nosso serviço pode conter links para outros sites que não são operados por nós. Se você clicar em um link de terceiros, será direcionado para o site desse terceiro. Recomendamos fortemente que você revise a Política de Privacidade de cada site que visitar.

7. Alterações a Esta Política de Privacidade

Podemos atualizar nossa Política de Privacidade periodicamente. Notificaremos você sobre quaisquer alterações publicando a nova Política de Privacidade nesta página.
